 Week 5 - React Hook Form Registration Project

Overview
This project is a simple Registration Form built with React using React Hook Form and Yup for validation.  
The goal of this assignment is to practice form handling, validation, and user feedback in React.

______________

Features

- Full Name: Required, minimum 3 characters
- Email: Required, must be a valid email format
- Password: Required, minimum 8 characters, must include at least 1 number
- Confirm Password: Required, must match Password
- Terms & Conditions: Checkbox, must be checked to submit
- Form Validation: Powered by Yup with React Hook Form
- Error Messages: Displayed under each invalid field
- Successful Submission: Displays "Registration Successful!" and logs form data in console
- Extra Features:
  - Show/Hide Password
  - Password Strength Indicator (Weak / Medium / Strong)

_______________


Installation & Running

1. Clone the repository:
```bash
git clone (https://github.com/setayeshazizi/react-hook-form-validation)

2. Install dependencies:
npm install

3. Start the development server:
npm start

_______________

Usage
 • Fill in all required fields correctly.
 • Password must contain at least 8 characters and 1 number.
 • Confirm password must match the password.
 • Accept Terms & Conditions to enable submission.
 • On successful submission, a success message appears and the form data is logged in the console.
